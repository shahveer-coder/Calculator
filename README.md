# 🧮 Python Calculator

A simple Calculator built using Python.

## 📖 Description

This calculator performs basic mathematical operations using a menu-driven program.

It is built with Python using loops, conditions, and user input.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Division by Zero Validation
- ❌ Invalid Choice Handling
- 🔁 Menu-Driven Program
- 🚪 Exit Option

## 🛠️ Technologies Used

- Python

## 🚀 How to Run

1. Download the project.
2. Open the project in VS Code.
3. Run the Python file.
